My Projects

Some examples of my work

Global Population Insights

This project aimed to create an interactive dashboard in Power BI to analyze global demographic trends. Users can explore key indicators such as population, fertility rate, median age, and net migrations from various perspectives: by continent and by country. This project allows for a better understanding of current global demographic dynamics and their future projections.

Methodology:

  • Global Overview: Presents an overview of global population trends, focusing on population growth, fertility rate, and median age projections up to 2050.
  • By Country/Continent: Provides a more detailed analysis, comparing different indicators across geographical regions.

Tools and Technologies Used:

  • Power BI: Data processing, visual design, and dashboard construction.
  • Data Sources: Demographic data from public databases (e.g., United Nations).

Data Scientist (Intern)

In The Memory, Paris, France

April 2024 - September 2024

In the context of my six-month final internship at In The Memory, I joined the Data team. My role was to develop a model capable of estimating the revenue generated by promotional products based on various characteristics entered by users, such as the period, product category, and discount percentage.

My Objectives:

  • Create a machine learning model for Carrefour to accurately estimate the revenue generated by promotional products.

Tasks Performed:

  • Database creation using Spark SQL to centralize all relevant information on product promotions.
  • Development of key indicators related to promotions (Seasonality Index, Geographic Index, Price Elasticity of Demand) to enrich analysis.
  • Implementation of various machine learning models on Databricks to estimate revenue generated by promotional products in units of need (UB).

Additional Missions:

  • Data audits for Carrefour to ensure data compliance and consistency.
  • Ad hoc studies for Carrefour and Auchan, using Spark SQL and Pandas to provide relevant insights.

Results:

  • Delivery of a machine learning model to predict promotional revenue.
  • More accurate predictions for better evaluation of return on investment.

Technical Environment:

  • Languages: Python
  • Platform: Databricks (datalake)
  • Technologies: Spark SQL, Pandas
  • Machine Learning Libraries: Scikit-Learn, TensorFlow, MLlib
  • Visualization Tool: Power BI
